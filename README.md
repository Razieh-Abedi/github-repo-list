GitHub Most Starred Repositories (Last 10 Days)
This web app displays the most starred GitHub repositories created in the last 10 days. The repositories are fetched from GitHub's API and displayed in a responsive, user-friendly interface. New repositories are loaded as the user scrolls down the page.

![Demo](demo.gif)

Features
Most Starred Repos: Displays repositories created in the last 10 days, sorted by stars.

Repository Details: For each repo, you will see the name, description, star count, and the owner's username and avatar.

Infinite Scroll: As you scroll to the bottom, new repositories are loaded (pagination).

Responsive Design: The app is designed to be fully responsive and optimized for all screen sizes.

Technologies Used
React.js: For building the frontend UI.

Axios: For making HTTP requests to GitHub's API.

CSS/SCSS: For styling the components and layout.

GitHub API: Fetching repositories based on specific criteria (created in the last 10 days, sorted by stars).
